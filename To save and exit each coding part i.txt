# GitCollab Project

This is a collaborative project repository created using Git Bash.  
It contains multiple files in different languages and formats to demonstrate teamwork and version control with Git.

## Files Included
- **Main.java** → Prints a greeting message in Java.
- **script.py** → Prints a greeting message in Python.
- **styles.css** → Adds a simple background color to the webpage.
- **index.html** → Displays a greeting message styled with CSS.
- **app.js** → Logs a greeting message in the browser console.
- **data.json** → Stores sample JSON data about the project.
- **README.md** → Provides documentation and description of the project.

## Purpose
The purpose of this project is to:
- Practice using Git and Git Bash for collaboration.
- Learn how to manage different types of files in a single repository.
- Demonstrate basic functionality in Java, Python, CSS, HTML, JavaScript, and JSON.
